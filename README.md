# Gnoke · Council

**A multi-AI deliberation thread. One HTML file. No backend. No dependencies. No server.**

You are the moderator. Claude, Gemini, GPT-4o, and Grok are the voices. You decide who speaks and when. You copy their context. You chair the room.

---

## What It Is

Not a chatbot. A coordinated deliberation between AI models with a human in the chair.

Switch identity to any AI, relay their response into the thread, and use **Copy context** to grab a perfectly filtered transcript — each AI only sees what others said, never their own words — ready to paste into any real AI chat.

Built as part of the [Gnoke Suite](https://dev.to/edmundsparrow) — a browser-native OS where tabs are processes, not pages.

---

## How To Use It

1. Open `index.html` in any modern browser
2. Post as **You** to open the discussion
3. Switch to **Claude** — copy context — paste into Claude — get response — relay it back
4. Repeat for Gemini, GPT-4o, Grok in any order you choose
5. You chair. They deliberate.

No API key. No installation. No build step. Works offline.

---

## The Copy Logic

Each AI's copy button filters out their own messages and returns everyone else's — so every member always has a fresh, relevant context to hand off. The moment anyone posts, everyone else's copy updates instantly.

For long sessions: summarise the thread yourself and post it as a Brief before copying. You are the intelligence.
